STEP 3: DEPLOY TO DIGITALOCEAN (Recommended)
---------------------------------------------
Time: 30 minutes | Cost: $6/month

1. Create account: https://www.digitalocean.com/
2. Create Droplet:
   - Ubuntu 22.04 LTS
   - $6/month (1GB RAM)
   - Choose datacenter
   - Create

3. SSH into droplet:
   ssh root@YOUR_DROPLET_IP

4. Run deployment:
   curl -fsSL https://raw.githubusercontent.com/Hammton/AI-employee/main/deploy_vps.sh -o deploy.sh
   chmod +x deploy.sh
   ./deploy.sh

5. Enter when prompted:
   - Repo: https://github.com/Hammton/AI-employee.git
   - OPENROUTER_API_KEY: (paste)
   - COMPOSIO_API_KEY: (paste)
   - MEM0_API_KEY: (paste)
   - USER_PHONE: +1234567890

6. Wait 5-10 minutes

7. Connect WhatsApp:
   systemctl stop wpp-bridge
   cd /root/pocket-agent/wpp-bridge
   DISPLAY=:99 node index.js
   (Scan QR code)
   (Press Ctrl+C)
   systemctl start wpp-bridge

8. Test:
   Send WhatsApp: "Hello! Are you working?"

TROUBLESHOOTING
---------------
WhatsApp won't connect:
  rm -rf /root/pocket-agent/wpp-bridge/tokens/*
  systemctl restart wpp-bridge

Services won't start:
  journalctl -u pocketagent -xe
  journalctl -u wpp-bridge -xe

Can't SSH:
  ufw allow 22
  systemctl restart sshd
